Prework Study Guide

HTML

  • The head element contains information(metadata) about the webpage
  • The body element represents the visible content shown to the user.
  • Meta tags are used by the browser and are not visible to the user
  • Use semantic elements to make your webpage as accessible as possible
  • It is important to use "alt" tags within img elements for functionality with screen readers and for higher search optimizations
  • Viewport tags are important on every webpage because they control how a page will be displayed on different devices

CSS

  • Stands for Cascasding Style Sheets and is a way to style our HTML and make it visually appealing.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Starting your code with "." specifies you want to use a selector.

Git

  • Bash which is a CLI (command line interface)
  • Used to navigate through the terminal and access directories and branches to work on
  • Connectivity with VS Code allows seamless integration which allows editing work thru a GUI and then viewing on a live server
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
